# Academia Barista & Beverage — Site Pronto (HTML/CSS)

Arquivos incluídos:
- `index.html` — página principal (single-page) pronta para usar.
- `styles.css` — folhas de estilo separadas.
- `README.md` — instruções rápidas de uso.

## Como usar localmente
1. Descompacte o ficheiro ZIP.
2. Abra `index.html` num navegador (duplo clique).

## Como publicar gratuitamente
### GitHub Pages (recomendado)
1. Crie um repositório no GitHub e envie os arquivos (`index.html` e `styles.css`).
2. No repositório, vá a **Settings → Pages** e selecione a branch `main` e a pasta `/ (root)`.
3. O site ficará disponível em `https://<seu-usuario>.github.io/<seu-repo>/` em alguns minutos.

### Netlify
1. Crie conta gratuita em Netlify.
2. Faça upload da pasta do site (drag & drop) ou conecte ao repositório GitHub.
3. Publica automaticamente; fornece domínio gratuito `*.netlify.app`.

## Personalização (substituir placeholders)
- Troque `seu-email@exemplo.com` por seu e‑mail real no `index.html`.
- Troque o link WhatsApp `https://wa.me/351000000000` pelo seu número (formato internacional).
- Substitua as "image-placeholder" por imagens reais (subir para Netlify ou GitHub e apontar src).
